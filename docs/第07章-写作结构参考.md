本文档提炼第 7 章定稿的写作结构、实操格式与细节，供第 8～11 章（第三篇「五类人群」）及后续章节参考。

## 二、扣子编程实操小节统一格式

- **标题**：`#### 【实操：在扣子编程上创建 XXX Agent】`
- **入口**：在扣子编程里… / 打开 https://code.coze.cn …
- **可选两种写法**：
  - **完整三步**（7.2.1）：第一步 登录并新建项目（1. 2. 3.）；第二步 用自然语言描述能力（可贴 JD 或给「描述示例」）；第三步 生成并试运行（测试语 + 不符合则对话补充约束）。
  - **精简版**（7.2.2、7.2.3）：直接「在扣子编程里同样选择『新建项目』→ 智能体」+ 配置要点表（若有）+ **描述示例**（引用块）+ 强调平台关键词（如「需要图像生成能力」）+ 配图 + 试运行一句 + 收尾「和 7.2.1、7.2.2 一样，核心都是把需求用自然语言说清楚」。
- **描述示例**：用 `>` 引用块；多段时每段前都加 `>`；内容要可直接复制使用（输入、输出、约束、格式）。
- **配图**：`![扣子编程-描述XXX能力](images/chapterN_coze_xxx.png)`，路径相对当前文件 `images/`。
- **平台名**：统一「扣子编程」，官网写为 `https://code.coze.cn`（无末尾斜杠）；正文可不写「Coze 2.0」除非必要。

## 三、表格与格式

- **表格**：表头用 `| 列1 | 列2 |`，单元格内强调用 `**人**`、`**Agent**`；产出物行可用 `—` 或 `产出：XXX`。
- **心法**：`**【心法】**` 单独起段，段末收束，不接在实操步骤里。
- **回扣前文**：用「还记得第 X 章…」「这正是第 X 章…」；大段用 `> **💡 回扣方法论**：…` 引用块，并分条 1. 2. 3.。
- **人物**：小陈贯穿；痛点、改造、数据、资产三件套都通过他推进；对标账号等可写具体名字（如 秋芝2046、朋克周、赛文乔伊）增强真实感。

## 四、JD / 配置与 Bug 表

- **岗位说明书**：可用代码块 ```markdown 贴完整 JD（# Role / Profile / Goals / Constraints / Input / Workflow / Output Format）；对标账号等可具体化。
- **配置要点表**：列 配置项 | 内容；输入/输出/结构/风格/字数/红线等。
- **打补丁表**：错误现象 | 原因分析 | 修正方案（在扣子/提示词里加：…）；修正方案列写具体加的一条话术，便于读者照做。

## 五、配图与图片路径

- **概念图**：十步流程图、人机分工图、日更工作流全景、改造前后对比；文件名 `chapterN_xxx.png`，路径 `images/chapterN_xxx.png`（相对当前 md 所在目录）。
- **实操截图占位**：扣子编程新建项目、描述能力、试运行、脚本配置、图片 Agent、打补丁等；`images/chapterN_coze_xxx.png`，由作者实际截图后放入。

## 六、第 7 章定稿中的具体细节（易忘）

- 开篇金句不加粗：「小陈以为自己缺的是灵感。其实他缺的是一条生产线。」
- 7.2.1 第二步：可写「把上面那份岗位说明书粘贴到输入框中」并补充「当然你也可以用自然语言描述」+ 单独「描述示例」引用块。
- 7.2.2 输入写「选题+角度」；输出内容含「合适的标题、完整的脚本、发布时需要填写的标签」；描述示例单独成块。
- 7.2.3 图片 Agent：用途一（配图）、用途二（封面）、约束；试运行示例可写具体标题如《OpenClaw：GitHub 星标 167k 的 AI 打工人…》；无「第一步」「第二步」小标题，一气呵成。
- 7.3 不写「进阶实操：工作流串联」长段，只保留「💡 回扣方法论」引用块（五步法落地）。
- 7.4.2 智能体列表：选题智能体、脚本智能体、**配图智能体**（输入标题，输出 3 版封面或配图）；打补丁表用「在提示词里加：…」。
- 资产三件套：选题、脚本、**封面**三个智能体；「助理登录**账号**就能用」。
- 本章结语：不单独提第 6 章试跑；「在扣子上搭出来」保留；「扣子编程」统一。

## 七、后续章节（第 8～11 章）可复用要点

- 换人物、换赛道（电商/教育/职场/创业），**结构不变**：开篇人物+痛点 → 7.1 拆步骤 → 7.2 分工+三类 Agent+扣子编程实操 → 7.3 标准工作流时间表+回扣 → 7.4 案例（初始/改造/资产三件套）→ 本章结语。
- 每章至少三个 **【实操：在扣子编程上创建 XX Agent】**，对应该行业核心环节；格式与 7.2.1 / 7.2.2 / 7.2.3 对齐。
- 平台统一：扣子编程（https://code.coze.cn）；描述示例、试运行语、打补丁话术按该行业改写。
- 配图：概念图（流程图/分工/时间线/对比）+ 扣子编程截图占位，路径一律 `images/chapterN_xxx.png`。
